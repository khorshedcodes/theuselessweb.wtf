# 🌀 theuselessweb.wtf

The open-source, community-driven platform for the internet's most wonderful nonsense.

## ✨ Features

- **Isolated Sandbox:** Secure `<iframe>` environment for user submissions.
- **Shuffle Engine:** Intelligent randomization with weighted categories.
- **Global Stats:** Real-time tracking of "Total Wasted Time" across the planet.
- **Viral Sharing:** Built-in sharing tools and dynamic social metadata.
- **UX Polish:** Retro sound effects, shuffling overlays, and milestone celebrations.

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

## 🛠️ Maintenance

### Health Checks
Keep the registry clean by checking if external sites are still online:
